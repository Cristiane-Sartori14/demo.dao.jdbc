# 🗄️ Acesso a Dados com JDBC (DAO)

> 💻 Projeto desenvolvido durante o curso "Java COMPLETO - Programação Orientada a Objetos + Projetos" (Udemy)

---

## 🏢 Sobre o projeto

Aplicação desenvolvida em Java com o objetivo de demonstrar o acesso a banco de dados utilizando JDBC, aplicando o padrão de projeto DAO (Data Access Object).

O projeto simula operações de persistência de dados, permitindo a manipulação de informações em banco de dados através de uma estrutura organizada e desacoplada.

---

## ⚙️ Funcionalidades

- Conexão com banco de dados utilizando JDBC  
- Operações CRUD (Create, Read, Update, Delete)  
- Implementação do padrão DAO  
- Separação de responsabilidades (camadas)  
- Manipulação de dados com SQL  

---

## 🛠️ Tecnologias utilizadas

- Java  
- JDBC  
- MySQL  
- SQL  
- Programação Orientada a Objetos (POO)
